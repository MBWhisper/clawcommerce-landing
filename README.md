# AURA Store 🕊️ | Premium React E-Commerce Template

A high-end, production-ready React foundation designed for luxury brands and modern storefronts. Built for performance, SEO, and visual excellence.

## ✨ Key Features
- **Luxury UI/UX**: Hand-crafted design specifically for premium brands.
- **Next-Gen Tech**: Built with React, Vite, and Smooth Micro-animations.
- **SEO Optimized**: Standard SEO tags, canonical URLs, and semantic HTML.
- **Multi-Brand Ready**: Easily switch between themes and color palettes.
- **RTL Support**: Built-in support for Arabic and RTL languages.
- **Mobile First**: Pixel-perfect responsiveness on all devices.

## 🚀 Quick Start
1. **clone or download**: `git clone <repository-url>`
2. **Install dependencies**: `npm install`
3. **Run Dev Server**: `npm run dev`
4. **Build for Production**: `npm run build`

## 🛠 Project Structure
- `src/components`: Reusable UI elements (Navbar, Footer, ProductCards).
- `src/pages`: Main storefront views (Home, Collections, About).
- `src/assets`: High-resolution graphics and design tokens.
- `public/css`: Global design system and component-specific styles.
